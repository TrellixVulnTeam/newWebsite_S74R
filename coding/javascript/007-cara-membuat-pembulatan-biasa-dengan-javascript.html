Metode Pembulatan adalah metode matematika yang mana menggeser nilai suatu bilangan ke bilangan bulat tertentu. Salah satu jenis pembulatan adalah pembulatan biasa atau pembulatan ke nilai terdekat. Pembulatan ini menggeser bilangan ke bilangan bulat terdekat. Misalnya 1.2 digeser menjadi 1, atau 3.6 digeser menjadi 4.

Berikut ini cara membuat pembulatan biasa atau pembulatan ke nilai terdekat dengan menggunakan javascript.

Berikut ini hasil dari pembulatan beberapa nilai dengan menggunakan Math.round():

nilai hasil
0.3 0
0.4 0
0.45 0
0.49 0
0.5 1
0.7 1
4.4 4
4.5 5

Dari hasil tersebut dapat dilihat bahwa nilai desimal yang lebih besar atau sama dengan 0.5 akan dibulatkan menjadi 1, selain itu misalnya 0.49 dibulatkan menjadi 0.

Misalnya lagi pada nilai 4.4, nilai desimal 0.4 dibulatkan menjadi 0 sehingga nilai menjadi 4. Pada nilai 4.5, nilai desimal 0.5 dibulatkan menjadi 1 sehingga nilai menjadi 4+1 = 5.
