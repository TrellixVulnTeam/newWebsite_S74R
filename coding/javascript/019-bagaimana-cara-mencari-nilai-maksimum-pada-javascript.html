Nilai maksimum adalah satu nilai paling tinggi dari serangkaian atau sekumpulan nilai yang diberikan. Misalnya jika diberikan serangkaian nilai dari 30, 45, 33, 21, 20 maka nilai paling tinggi adalah 45. Cara mencari nilai maksimum pada javascript tidak jauh berbeda dengan cara mencari nilai minimum.
