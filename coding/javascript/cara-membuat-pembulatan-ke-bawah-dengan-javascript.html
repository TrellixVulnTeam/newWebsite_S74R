Selain pembulatan biasa, pembulatan ke bawah juga bisa dilakukan di javascript. Pembulatan ke bawah adalah pembulatan dimana menggeser nilai suatu bilangan menjadi bilangan bulat dibawah nilai tersebut. Pembulatan ke bawah juga bisa dijelaskan dengan menghilangkan nilai desimal pada bilangan sehingga hanya menyisakan bilangan bulatnya saja. Misalnya 0.4 menjadi 0, dan 0.8 menjadi 0.

Berikut ini cara membuat pembulatan kebawah dengan menggunakan javascript.

Berikut ini hasil dari pembulatan beberapa nilai dengan Math.floor():

nilai hasil
0.3 0
0.4 0
0.45 0
0.49 0
0.5 0
0.7 0
4.4 4
4.5 4

Dari hasil di atas dapat dilihat bahwa pembulatan jenis ini membulatkan nilai desimal menjadi 0, atau hanya menghilangkan nilai desimal pada bilangan. Misalnya dari 0.45 menjadi 0, atau dari 4.5 menjadi 4.
