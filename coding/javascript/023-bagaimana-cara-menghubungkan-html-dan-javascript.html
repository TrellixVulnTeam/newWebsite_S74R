HTML adalah bahasa yang digunakan untuk menentukan apa saja isi dari sebuah halaman website, seperti header, paragraf, atau bahkan input dan "button".

Sementara Javascript adalah bahasa pemrograman yang digunaan untuk mengatur perilaku dari sebuah website. Lalu bagaimana cara menghubungkan keduanya.

Fungsi getElementById() adalah salah satu fungsi yang bisa digunakan untuk menghubungkan element HTML dengan javascript. Fungsi ini digunakan untuk mengedit isi element dengan "id" tertentu.

Pada halaman HTML, "id" adalah hal yang harus unik atau berbeda satu sama lain. Anda tidak diperbolehkan menggunakan "id" yang sama berulangkali dalam website. Anda hanya diperbolehkan menggunakannya satu kali dalam satu halaman website. Jadi, pada saat anda menggunakan fungsi getElementById() anda hanya bisa mengubah satu element dengan "id" tersebut.
