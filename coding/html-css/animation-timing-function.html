Animation Timing Function merupakan kurva kecepatan suatu animasi. kurva ini bertujuan untuk mengatur kecepatan animasi awal sampai akhir. Ada yang lambat di awal kemudiaan semakin cepat, ada yang cepat di awal yang kemudian semakin lambat, ada juga yang lambat di awal dan di akhir tapi cepat di pertengahan.

  • ease : lambat di awal, kemudian cepat, dan kembali melambat diakhir
  • ease-in : lambat di awal.
  • ease-out : lambat di akhir.
  • ease-in-out : lambat di awal dan di akhir.
  • linear : kecepatan sama dari awal hingga akhir.
